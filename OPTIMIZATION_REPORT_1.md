# 优化报告 #1 - 2026-03-21 15:37

## 发现的问题

### 🔴 需要修复
1. **resizeEvent 无防抖** — 拖拽窗口边缘时 `_load_current()` 每帧都调用，500 页漫画会导致严重卡顿
2. **eventFilter 中重复 import QEvent** — 每次鼠标事件触发都执行 import，应提升为模块级
3. **LRUCache.__contains__ 竞态** — `PreloadThread` 中 `if i not in cache` 释放锁后立即 emit，但加载完成时 cache 可能已满并淘汰了该 key
4. **全屏切换不隐藏工具栏** — 全屏模式下工具栏仍然占空间，影响沉浸感
5. **未使用的导入** — `QIcon`, `QTimer` 在 viewer.py 中导入但从未使用

### 🟡 可以改进
6. **缩略图对话框关闭时线程池仍在运行** — 应取消未完成任务
7. **双页模式最后一页右侧空白无提示** — 应显示"已是最后一页"提示
8. **对话框不支持 Escape 关闭** — 用户习惯 Escape 关闭弹窗
9. **长图模式下上下键不可用** — 应支持 ↑/↓ 滚动长图
10. **PreloadThread 缓存判断逻辑冗余** — 可简化为只在 emit 前检查
