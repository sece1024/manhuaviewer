# 优化报告 #2 - 2026-03-21 15:42

## 深度审查发现

### 🔴 架构 / 健壮性
1. **data_store.py 无异常恢复** — history.json 写入过程中断电/崩溃会导致数据丢失，应先写临时文件再 rename
2. **未处理的 QDir 隐患** — `QDir.entryList` 在文件夹被删除后返回空列表但不报错，用户可能困惑
3. **全屏 F11 与菜单栏快捷键冲突** — 全屏时菜单栏隐藏，F11 需要从 keyPressEvent 触发，但 keyPressEvent 不会被菜单拦截
4. **标签颜色不跟随主题** — 深色主题下标签文字仍然是深色
5. **SettingsDialog scale_factor 精度丢失** — int 除法 /100 导致小数位丢失（如 75% → 0.75 但 125% → 1.25 正好可以）

### 🟡 用户体验
6. **没有文件夹计数/过滤逻辑** — 打开一个混合文件夹（图片+其他文件）时，只显示图片但不提示有多少非图片文件
7. **翻页到边界无提示** — 到第一页/最后一页时无视觉反馈，用户不知道已经到底了
8. **缩略图无当前页高亮滚动** — 打开缩略图时不会自动滚动到当前页
9. **状态栏信息可以更丰富** — 可以显示文件大小、图片分辨率等
10. **右键菜单缺失** — 图片显示区没有右键菜单

### 🟢 代码质量
11. **viewer.py 仍有 759 行** — 可进一步拆分 `_create_menu_bar` 到单独方法或使用 UI builder 模式
12. **magic number** — 滚轮缩放系数 1.15、长图滚动步长 30、80 等应提取为常量
13. **test_data_store.py 中的 sys.path.insert 应移除** — 已经通过 pyproject.toml 配置了 src layout
